# Tus Derechos

Plataforma de educación legal accesible para jóvenes y público general en España. Información clara, sin tecnicismos, sobre tus derechos reales.

## 🎯 Objetivo

Ayudar a personas jóvenes a entender sus derechos legales en España sin necesidad de jerga complicada. **No ofrecemos asesoramiento legal personalizado**, solo información educativa general.

✅ **Optimizado para SEO** - Artículos enfocados en búsquedas reales de Google  
✅ **Diseño minimalista** - Fácil de leer, tipo startup legaltech  
✅ **Responsive** - Perfecto en móvil, tablet y desktop  
✅ **Sin frameworks pesados** - HTML + CSS + JavaScript puro  
✅ **Información práctica** - Casos reales, no teoría académica
